# Forza Horizon 6 刷技能點小助手 (Skill Farming Bot)

Language / 語言: [繁體中文 (Traditional Chinese)](README.md) | [English](README_EN.md)

這是一個專為《極限競速 Forza Horizon》系列設計的自動化刷技能點輔助工具。本工具透過 **OpenCV 圖像辨識（模板匹配）** 技術偵測遊戲狀態，並利用 Windows 底層的 **DirectInput 鍵盤與滑鼠模擬** 技術自動重新開始賽事，實現無人值守全自動刷技能點。

本專案配有現代暗色系圖形化介面（GUI）與自訂圖像裁剪校準工具，能完美相容於各種螢幕解析度與遊戲語系。

---

## 功能特點

- **DirectInput 模擬**：使用 Windows `SendInput` API，100% 繞過遊戲的防一般鍵盤輸入屏蔽，安全且穩定。
- **自訂模板校準**：內建「截圖裁剪工具」，點擊擷取即可直接在遊戲畫面中拖拉框選您畫面上的按鈕，解決解析度與語系不合問題。
- **全域快捷鍵**：在遊戲中隨時按下 `F10` 啟動自動化，按下 `F11` 立即停止，附帶 W 鍵安全釋放鎖定。
- **自動前進**：在賽事進行時會自動按住 `W` 鍵使車輛保持前進，賽事時間結束自動釋放。
- **智能啟動偵測**：當小助手啟動時，會自動辨識遊戲當前正處於「結算」、「確認重新開始彈窗」還是「準備起跑」畫面，並自動無縫對接下一步執行，無需從頭等待。
- **自由置頂開關**：下拉選單一鍵選擇目前電腦上運行的遊戲視窗，並可選取「視窗強制置頂」將遊戲固定顯示在畫面最前方（可隨時自由開啟或關閉，防止其它視窗遮擋辨識）。
- **背景維持聚焦指引**：在參數設定頁面提供「❓ 背景掛機指引」按鈕，點擊後會彈出詳細說明，教導使用者如何搭配免費軟體 DisplayFusion 的「防止視窗停用」功能或利用虛擬機（VM），在不進行任何 DLL 注入（確保 100% 防封號安全）的情況下安全實現背景掛機不凍結。
- **自動定時關閉**：提供定時終止選項（如 `1 小時`、`1.5 小時`、`2 小時` 等），啟動後介面會即時顯示倒數計時器，時間到時自動安全停止腳本，方便玩家控制掛機時長。

## 安裝與準備工作

在執行本工具前，請確認您已完成以下準備：

### 1. 安裝 Python 軟體
- 請前往 [Python 官方網站](https://www.python.org/) 下載並安裝 **Python 3.12** 或以上版本。
- **【重要】** 安裝時請務必勾選 **「Add Python.exe to PATH」** 選項。

### 2. 安裝 Python 依賴套件
- **一鍵自動安裝（強烈推薦）**：直接雙擊執行專案資料夾下的 **`install_requirements.bat`** 批次檔。它會自動呼叫系統命令列，**為您一鍵安裝此腳本運作所需的全部 Python 套件（包括 OpenCV、Pillow 與 pywin32）**，省去手動開啟終端機與輸入指令的步驟。
- **手動安裝**：如果您偏好手動安裝，可開啟命令提示字元（CMD）並執行以下指令：
```bash
pip install opencv-python pillow pywin32
```

### 3. 遊戲畫面設定
為了使截圖與辨識功能正常運作，請進入《極限競速》的畫面設定中：
- 將顯示模式設為 **「視窗模式」 (Windowed)** 或 **「無邊框視窗模式」 (Borderless Windowed)**。
- ⚠️ 請勿使用「獨占全螢幕」模式，否則腳本會抓取到黑畫面。

## 使用教學

1. **以管理員身分執行**：
   - 在專案目錄下的 `run_bot.bat` 檔案上點按滑鼠右鍵。
   - 選擇 **「以系統管理員身分執行」**（因為遊戲以高權限運行，輔助小助手也需相同權限才能模擬按鍵）。

2. **校準圖像模板（第一次使用必做）**：
   - 開啟小助手介面後，讓遊戲顯示在螢幕上。
   - 在小助手右側的「圖像匹配模板校準」中，依序點擊 **「擷取」** 按鈕：
     - **重新開始字樣**：在賽事結束、出現結算畫面時點擊擷取，然後在畫面的「重新開始」文字上**按住左鍵並拖曳框選**。
     - **確認選單「是」**：在按下重新開始後跳出的「是否確認重新開始」提示框中，框選「是」字樣（偵測到後會自動按下鍵盤 `Enter` 鍵進行確認）。
     - **開始賽事字樣**：在進入賽事準備畫面時點擊擷取，框選「開始賽事」字樣。
   - 框選完畢後，小助手上會顯示預覽圖，且文字變綠色即代表校準成功。

3. **調整賽事時間**：
   - 在左側的「參數設定項目」中，您可以輸入單局賽事所需的秒數（例如 `62` 秒）。
   - 設定完成後點擊 **「儲存設定」**。

4. **啟動與停止**：
   - 進入遊戲後，按下鍵盤上的 **`F10`** 鍵啟動自動化（或點擊介面上的「啟動腳本」）。
   - 想要結束或中途暫停時，按下 **`F11`** 鍵（或點擊介面上的「停止腳本」），車輛會立即停止，小助手進入安全停止狀態。

5. **自動購車模式使用方式 (Auto Car Purchase Mode)**：
   - **模式切換**：在小助手主畫面上的「選擇掛機模式項目」中勾選 **「自動購買車輛 (Lamborghini Revuelto)」**。
   - **完成模板校準**：切換至「圖像校準」分頁，依序點擊「擷取」並在遊戲中框選以下三個新模板：
     - **汽車展售中心 (`autoshow.png`)**：在車庫首頁時框選該按鈕。
     - **蘭博基尼商標 (`lambo_brand.png`)**：在車廠選單中框選「LAMBORGHINI」圖示。
     - **Revuelto卡片 (`revuelto.png`)**：在車款列表中框選「REVUELTO」車型卡片。
   - **開始購車**：進入遊戲車庫首頁，按下 **`F10`** 鍵啟動。小助手會自動偵測按鈕位置進行點擊、發送 `Backspace`、進入選單選取 Revuelto、選擇顏色、發送 `Enter` 完成購買、等待 10 秒動畫過場，最後發送 `Esc` 返回首頁並重複，實現全自動批次購車。

6. **自動點熟練度模式使用方式 (Auto Car Mastery Mode)**：
   - **模式切換**：在小助手主畫面上的「選擇掛機模式項目」中勾選 **「自動點熟練度」**。
   - **完成模板校準**：切換至「圖像校準」分頁，依序點擊「擷取」並在遊戲中框選以下四個新模板：
     - **我的車輛 (`my_cars_tile.png`)**：在車庫首頁時框選該按鈕。
     - **乘駕車輛 (`drive_car.png`)**：點擊車輛卡片後，在跳出的「乘駕車輛」按鈕上框選。
     - **升級套件與調校 (`upgrades_tuning.png`)**：在車庫主選單中框選該按鈕。
     - **車輛熟練度 (`car_mastery_button.png`)**：在升級選單中框選該按鈕。
   - **校準4x4技能樹網格**：在「圖像校準」分頁下方，點選「校準左上角」與「校準右下角」，分別在遊戲進入車輛熟練度畫面時，點選4x4技能樹最左上角與最右下角按鈕的中心位置。
   - **開始點選熟練度**：進入遊戲車庫首頁，按下 **`F10`** 鍵啟動。小助手會自動進入我的車輛、篩選蘭博基尼、依序點選未處理車輛並乘駕（第二次選取自動改用 Enter 避開滾動位移）、進入升級與熟練度頁面、依序解鎖6個技能點、返回車庫大廳並將處理車數加1，接著重複運作。

## 背景掛機與防凍結指南 (Background Idle & Anti-Freeze Guide)

由於《極限競速》系列（Forza Horizon）採用 DirectX 12 渲染與原始輸入 (Raw Input) 系統，當遊戲視窗失去焦點時（例如進行 Alt-Tab 切換），遊戲會自動暫停、降低幀率並暫停接受輸入。

### ⚠️ 安全性與輸入限制警告
1. **防封號安全設計**：要直接在程式碼層級欺騙遊戲繞過焦點檢測，必須將 DLL 注入遊戲進程中。然而，這會被遊戲的防作弊系統（如 Easy Anti-Cheat）檢測為外掛，導致**永久封號**。為此，本小助手**絕不進行任何危險的代碼注入**。
2. **鍵盤按鍵洩漏問題**：本工具使用 Windows 的 `SendInput` API 來模擬真實的物理鍵盤按鍵（以防止被遊戲阻擋）。`SendInput` 產生的按鍵**只會發送到目前系統的活動視窗**。如果您在掛機時在 Chrome 或其他聊天軟體中打字，腳本的 `W`、`X`、`Enter` 按鍵會直接打在您正在輸入的應用程式中，這將干擾您的正常操作。

### 💡 解決方案與最佳實踐

#### 方法 A：使用 DisplayFusion (適用於雙螢幕 / 僅觀看監控)
這是最常見的背景防凍結方法：
1. 確保遊戲設定為**「無邊框視窗模式」**。
2. 安裝 [DisplayFusion](https://www.displayfusion.com/) 軟體（免費版即可）。
3. 在 DisplayFusion 的「Settings -> Functions」中，尋找 **「Prevent Window Deactivation (防止視窗停用)」** 並為其分配一組快捷鍵（例如 `Ctrl + Alt + P`）。
4. 單擊遊戲視窗使其成為活動視窗，按下您設定的快捷鍵。
5. 之後，您可以將滑鼠移到第二個螢幕進行其他不使用鍵盤輸入的操作（例如看影片、滑鼠瀏覽網頁），遊戲將不會暫停或凍結。

#### 方法 B：使用 Windows 虛擬機 (最推薦，可邊掛機邊工作)
如果您需要在掛機時正常用鍵盤打字、工作或玩其他遊戲：
1. 使用 Windows 的 Hyper-V、VMware 或 VirtualBox 建立一台 Windows 虛擬機。
2. 確保為虛擬機開啟了 **GPU 3D 加速**（GPU 直通）以支持執行 DirectX 12 遊戲。
3. 在虛擬機內執行遊戲和本小助手。
4. 將小助手啟動後，您可以直接將虛擬機視窗縮小，這時主機的任何鍵盤輸入都不會影響到虛擬機內的腳本，實現完全的後台隔離掛機！

#### 方法 C：利用內建定時器離線掛機
如果不需要同時使用電腦：
- 可以在「參數設定」中開啟「自動停止定時」（可選 1 小時、1.5 小時等），並保持遊戲在前台焦點位置。掛機結束後小助手會自動停止，確保電腦省電。
